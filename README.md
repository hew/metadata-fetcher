## Metadata Fetcher

This project demonstrates how to retrieve metadata from various different endpoints.

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start retrieving metdata from diffrent endpoints.
